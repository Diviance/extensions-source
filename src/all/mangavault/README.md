## FAQ

### Why do I see no manga?
MangaVault is a self-hosted manga server extension.

### The MangaVault extension stopped working?
Make sure that your MangaVault server and extension are on the newest version.

### Can I add more than one MangaVault server or user?
Yes, currently you can add up to 11 different MangaVault instances to the app. The number of instances
available can be customized in extension settings.

### Why are EPUB chapters/books missing?
Mihon/Tachiyomi does not support reading text. EPUB files containing only images will be available
if your MangaVault server version supports Divina-compatible EPUB handling.

## Guides

### How do I add my MangaVault server to the app?
Go into the settings of the MangaVault extension (under Browse -> Extensions) and fill in your server
address and login details.
